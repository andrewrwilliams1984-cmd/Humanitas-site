Humanitas polished site package

Contents:
- index.html
- assets/favicon.svg
- assets/og-image.png

This is the final polished static site package for GitHub + Cloudflare Pages.

How to deploy:
1. Create a GitHub account
2. Create a new repository, for example: humanitas-site
3. Upload everything in this folder
4. In Cloudflare Pages, connect that GitHub repository
5. Use:
   - Framework preset: None
   - Build command: leave blank
   - Output directory: /
6. Deploy

Notes:
- No coding setup is required
- The site is static and self-contained
- Mobile navigation is included
- Proper SVG icons, favicon, and social share image are included
